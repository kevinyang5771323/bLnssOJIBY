# 简单学生管理系统 python782

本项目是一个基于Python语言和sqlite3数据库的简单学生管理系统。

## 项目概述

本系统致力于为教育机构提供一个便捷的学生信息管理工具。

## 技术栈

- 后端：Python
- 数据库：sqlite3

## 功能模块

- 学生信息管理
  - 学生信息录入
  - 学生信息查询
  - 学生信息修改
  - 学生信息删除
- 成绩管理
  - 成绩录入
  - 成绩查询
  - 成绩修改
  - 成绩删除

## 系统角色

- 管理员：负责系统的维护和学生信息的全面管理

## 运行环境

- 操作系统：兼容Windows、Linux和Mac OS
- Python版本：3.x

## 部署步骤

1. 安装Python环境
2. 克隆项目代码
3. 安装依赖库（sqlite3）
4. 运行系统

## 目录结构

```
简单学生管理系统
│
├── app.py            # 主程序
├── database          # 数据库相关文件
│   └── sqlite.db     # sqlite数据库文件
└── student.py        # 学生管理模块
```

## 核心亮点

- 简洁的界面设计，便于操作
- 使用sqlite3作为数据库，无需复杂配置，便于上手
- 完善的学生信息管理功能，满足日常管理需求
